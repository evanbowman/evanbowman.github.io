Skyland (2021-2024)

A realtime strategy game for Nintendo Gameboy Advance, developed in about twenty days for a competition, and then continuously revised for a number of years afterwards. Explore the skies, while building a fortress in the clouds. Download the latest ROM file here, or click the image to play an older version of the game in the browser. This site features a pre-release beta version of the final game.
