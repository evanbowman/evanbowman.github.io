SKYLAND (2021-2022)

A realtime strategy game for Nintendo Gameboy Advance, developed in about twenty days for a competition. Explore the skies, while building a fortress in the clouds. Download the latest ROM file here, or click the image to play an older version of the game in the browser. This site features a pre-release beta version of the final game.


Blind Jump (2020)

A computer game for the Nintendo Gameboy Advance. This highly polished game features vibrant, meticulously designed color palettes, fluid animations, and algorithmically generated level designs. Click the screenshot above run in the browser, or download the game here. Requires a Gameboy Advance and a flash cartridge to run.
