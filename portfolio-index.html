SKYLAND (2021)

A realtime strategy game for Nintendo Gameboy Advance, developed in about twenty days for a competition. Explore the skies, while building a fortress in the clouds. Download the latest ROM file here, or click the image to play an older version of the game in the browser.


Blind Jump (2020)

A computer game for the Nintendo Gameboy Advance. This highly polished game features vibrant, meticulously designed color palettes, fluid animations, and algorithmically generated level designs. Click the screenshot above run in the browser, or download the game here. Requires a Gameboy Advance and a flash cartridge to run.


LISP

Both SKYLAND and BlindJump, above, feature an embedded lisp interpreter. The Gameboy Advance hardware limits the fuctionality of the interpreter significantly; this LISP dialect does not include several notable features that one might expect from a LISP implementation, such as functional closures. A LISP-1, like Scheme, so functions and variables inhabit the same namespace. Like Emacs LISP, our interpreter uses dynamic variable scope.

Note that this standalone version of the interpreter includes only the LISP language and standard library, it does not include any of the script hooks into the SKYLAND or Blind Jump engines. While not practically useful, you may find the web-based interpreter to be fun to play around with nonetheless.

Both SKYLAND and Blind Jump serve a LISP REPL via UART from the Link Cable port, so on a physical console, you can plug the gba into a laptop with a 3.3v RS232 cable, and send commands.
